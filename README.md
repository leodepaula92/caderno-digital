# 📓 Meu Caderno Digital Cloud

O **Meu Caderno Digital** é uma aplicação web completa desenvolvida para auxiliar estudantes e desenvolvedores na organização de suas notas de estudo. O projeto simula um ambiente de Software as a Service (SaaS), onde cada usuário possui sua própria conta protegida e seus dados sincronizados em tempo real na nuvem.

Este projeto foi desenvolvido como parte do curso de **Desenvolvimento de Sistemas**.

---

## 🚀 Funcionalidades

-   **Sistema de Autenticação:** Cadastro e Login de usuários via Firebase Auth.
-   **Recuperação de Senha:** Fluxo automático de redefinição de senha por e-mail.
-   **Privacidade de Dados:** Cada usuário visualiza e gerencia apenas suas próprias notas.
-   **CRUD Completo:** Criar, Ler, Editar e Excluir notas de estudo.
-   **Tags e Matérias Dinâmicas:** Adicione múltiplas tags e matérias separadas por vírgula.
-   **Filtros Inteligentes:** Menu lateral que gera botões de filtro automaticamente com base nas matérias e tags cadastradas.
-   **Design Responsivo:** Interface limpa e organizada, focada na experiência do usuário (UX).

---

## 🛠️ Tecnologias Utilizadas

As seguintes ferramentas e tecnologias foram utilizadas na construção do projeto:

-   **HTML5 & CSS3:** Estrutura e estilização personalizada com foco em acessibilidade.
-   **JavaScript (ES6+):** Lógica de manipulação de dados e integração com API.
-   **Firebase Realtime Database:** Banco de Dados NoSQL para persistência em nuvem.
-   **Firebase Authentication:** Gestão segura de usuários.
-   **GitHub Pages:** Hospedagem gratuita da aplicação.
