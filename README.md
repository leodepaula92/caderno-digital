# 📓 Meu Caderno Digital Cloud

O **Meu Caderno Digital** é uma aplicação web completa desenvolvida para auxiliar estudantes na organização de suas notas de aula. O projeto funciona como um sistema na nuvem, onde cada usuário possui sua conta individual e seus dados são sincronizados em tempo real.

---

## ✨ Funcionalidades

-   **Sistema de Login e Cadastro:** Segurança via Firebase Authentication.
-   **Recuperação de Senha:** Envio automático de e-mail para redefinição de acesso.
-   **Privacidade Total:** Cada usuário acessa apenas suas próprias notas através do seu UID.
-   **CRUD Dinâmico:** Criar, visualizar, editar e excluir notas instantaneamente.
-   **Organização por Matérias e Tags:** Suporte a múltiplas tags separadas por vírgula.
-   **Filtros Automáticos:** Menu lateral inteligente que gera filtros com base no que você cadastrou.
-   **Persistência em Nuvem:** Seus dados salvos no Google Firebase, acessíveis de qualquer dispositivo.

---

## 🛠️ Tecnologias Utilizadas

-   **HTML5 & CSS3:** Interface moderna e responsiva com foco em UX.
-   **JavaScript (ES6+):** Lógica de filtros, manipulação de DOM e integração Cloud.
-   **Firebase Realtime Database:** Banco de dados NoSQL para sincronização em tempo real.
-   **Firebase Authentication:** Gestão segura de identidades e e-mails.
-   **GitHub Pages:** Deploy e hospedagem da aplicação.
